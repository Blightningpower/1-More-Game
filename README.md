# 1 More Game

Een mobiele party-fitness game waarin spelers steeds één herhaling meer doen totdat er per oefening één winnaar overblijft.

## Eerste versie

- Spelers toevoegen
- Oefeningen kiezen of zelf toevoegen
- Alfabetische beurtvolgorde
- Grote beurt-teller
- Knoppen voor voltooid en opgegeven
- Winnaar per oefening
- Eindscore over alle oefeningen

## Lokaal starten

```bash
npm install
npm start
```

Scan daarna de QR-code met Expo Go op je telefoon.
